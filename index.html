Pixlpals

By Pixltoshi and Degentralabs



So I decided to create an interactive ordinal builder using recursion strictly for fun and because I needed to test a few things.. At first I was going to add some well known pre-inscribed collections to mess with like punks, apes, azuki, etc.. but as I was building this I decided to make some original (derivative) art for the experiment. The reason being I want to know who participates so that I can keep in contact moving forward and because I wouldn't be able to index those other collections if thats what I decide to do. There have been several recurive drops or attempts to frontrun collections but we haven't seen any that made the process as simple as building the ordinal you want, pressing build and inscribing it. Whenever you're ready just click the "Build" button below to get started.
